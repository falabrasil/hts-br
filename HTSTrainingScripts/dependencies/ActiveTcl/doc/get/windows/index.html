Installing ActiveTcl for Windows

Requirements

  • 35 MB for the final install
  • Windows 2003 or later

Installation

ActiveTcl is distributed on Windows as an executable installer (.exe file).

  1. Download the ActiveTcl AS Package for Windows (ActiveTcl-<version>-<platform>.exe).
  2. Double-click the installer file to extract the contents and run the installation.
    • It is strongly recommended that you start the installer by a right-clicking it and selecting Run as Administrator from the context menu.
    • Note further that it is necessary to save the installer to the local disk before running it, contrary to any objections made by recent versions of Internet Explorer. The installer cannot be run from IE.
